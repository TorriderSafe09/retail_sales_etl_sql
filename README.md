# Pipeline ETL de Ventas Minoristas (MySQL)

## Descripción general

Este proyecto demuestra un flujo de trabajo completo de ETL (Extraer, Transformar, Cargar) utilizando MySQL.

El pipeline incluye:

- Ingesta de datos en bruto
- Limpieza de datos
- Detección de duplicados
- Manejo de valores nulos
- Creación de la capa de negocio
- Modelado relacional usando claves primarias y foráneas
- Consultas analíticas

## Tecnologías

- MySQL
- MySQL Workbench
- SQL

## Estructura del proyecto

- Capa de datos en bruto
- Capa de preparación (Staging)
- Capa de negocio
- Consultas para reportes
